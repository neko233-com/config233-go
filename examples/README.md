# Config233 Examples

This directory contains example code demonstrating how to use the config233 library.

## Examples

### example_usage.go
Basic usage example showing how to use the config233 library to load and manage configuration files.

### manager_example.go
Example demonstrating the configuration manager features.

### logr_example.go
Example showing how to integrate with the logr logging interface.

### validation_demo.go
Demo of configuration validation features.

### excel_validation.go
Example of Excel file validation and loading.

## Running Examples

To run any example:

```bash
cd examples
go run example_usage.go
```

## Note

These files are for demonstration purposes only and are not part of the public API of the config233 library.
